# 文字生长器 - 基于语义的动态文字艺术

一个创新的交互式Web应用，让文字根据语义和情感"生长"成具象化的视觉形态。

## 🌱 核心概念

用户输入一段文字（如"春天"），系统根据文字的语义和情感，让文字"生长"成具象化的形态：

- **"春天"**：文字生长出花瓣、嫩芽，笔画变成藤蔓
- **"孤独"**：文字逐渐散开、笔画变细，背景出现飘落的碎片
- **"激情"**：文字燃烧、跳动，笔画带有火焰效果
- **"温柔"**：文字边缘柔和，带有爱心和气泡效果
- **"梦幻"**：文字带有星光和云雾效果，若隐若现

## 🎨 功能特性

### 1. 语义识别
- 自动识别输入文字的语义
- 匹配相应的视觉生长效果
- 支持多种情感和主题

### 2. 动态生长
- 四个生长阶段：种子 → 发芽 → 生长 → 绽放
- 每个阶段都有独特的视觉效果
- 平滑的动画过渡

### 3. 语音交互
- 录音功能捕捉语音节奏
- 根据语音强度调整生长速度
- 实时音频可视化

### 4. 交互控制
- 播放/暂停生长动画
- 调整生长速度
- 重置生长过程
- 手动选择效果模式

## 🚀 技术栈

- **Vite** - 构建工具

## 🎮 使用方法

1. **输入文字**：在输入框中输入想要生长的文字
2. **选择效果**：从预设的语义效果中选择，或让系统自动匹配
3. **控制生长**：使用播放/暂停按钮控制生长动画
4. **语音控制**：开启录音，根据语音节奏影响生长速度
5. **观察变化**：观看文字随生长阶段的变化效果

## 🎯 设计理念

### 语义可视化
将抽象的文字含义转化为具体的视觉元素，让用户"看到"文字的情感。

### 渐进式生长
模拟自然生长过程，从简单到复杂，让用户见证文字的演变。

### 多感官交互
结合视觉、听觉（语音）和交互控制，创造沉浸式体验。

## 📱 响应式设计

应用完全响应式，支持：
- 桌面端（大屏幕）
- 平板端（中等屏幕）
- 移动端（小屏幕）
